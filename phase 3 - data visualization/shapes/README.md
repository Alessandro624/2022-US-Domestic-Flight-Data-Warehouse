# Tableau Shapes

This folder contains the custom shape assets used by the Tableau workbook.

## Files

| File | Purpose |
| --- | --- |
| [airplane_icon_126136.png](airplane_icon_126136.png) | Airplane shape used in flight-oriented charts. |
| [eye.png](eye.png) | Eye shape used as an attention/inspection marker in dashboard elements. |

## Installation

Copy the PNG file into a custom folder inside the local Tableau shapes
repository:

```text
Documents/My Tableau Repository/Shapes/Departures Shape/
```

Then open Tableau and select `Reload Shapes` from the shape palette if the icon
is not visible immediately.

The folder name must remain `Departures Shape`, because this is the custom shape
palette name used by the workbook.
