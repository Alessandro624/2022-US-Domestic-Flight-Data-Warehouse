# Cleaned Database

## Overview

This directory contains the cleaned version of the reconciled database. The files are produced after data quality assessment, missingness analysis, outlier review, and cleaning transformations.

The cleaned layer preserves the same broad relational structure as the reconciled database, but it applies the project cleaning rules before the data is transformed into the dimensional warehouse.

## Expected Directory Structure

```text
data/
└── 3-cleaned database/
    ├── Active_Weather.csv
    ├── Aircrafts.csv
    ├── Cancellation.csv
    ├── Carriers.csv
    ├── Flights.csv
    ├── Stations.csv
    └── Weather_Observations.csv
```

## Generation

This layer is generated by the cleaning workflow in [5-Cleaning Pipeline.ipynb](../../phase%202%20-%20etl/5-Cleaning%20Pipeline.ipynb).
