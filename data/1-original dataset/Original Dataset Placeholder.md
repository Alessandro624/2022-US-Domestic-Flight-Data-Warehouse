# Original Dataset: 2022 US Airlines Domestic Departure Data

## Overview

This dataset contains comprehensive information about US domestic airline flights in 2022, including flight operations, weather conditions, aircraft details, and cancellation records.

**Source:** [Kaggle - 2022 US Airlines Domestic Departure Data](https://www.kaggle.com/datasets/jl8771/2022-us-airlines-domestic-departure-data/)

---

## Download Instructions

### Steps

1. **Access the dataset** on Kaggle using the link above
2. **Click "Download"** button on the dataset page
3. **Extract the downloaded ZIP** file
4. **Place the CSV files** into this directory: `data/1-original dataset/`

---

## Expected Directory Structure

After extraction, your directory should contain exactly these files:

```text
data/
└── 1-original dataset/
    ├── ActiveWeather.csv
    ├── Cancellation.csv
    ├── Carriers.csv
    ├── CompleteData.csv
    └── Stations.csv
```
